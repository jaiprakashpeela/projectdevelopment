## Now pulling the Code from Git Repo 
## 1. Clone the source code

```sh
$ git clone https://github.com/jaiprakashpeela/projectdevelopment.git
$ cd website
```
### 1. Install Node Modules
```sh
$ npm install
```
### 2. Build the Project
```sh
$ npm run build 
```
### 3. Run Test
```sh
$ npm run test
```
### 4. Start
```sh
$ npm start server
```
### 5. open the link in any browser 
```sh
http://localhost:8080/
```
